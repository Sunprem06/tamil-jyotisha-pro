
There is no separate "Connect to GitHub" button inside the Project settings page shown in your screenshots.

Where to find it instead:

Desktop
1. Click "Go back" from Project settings.
2. In the left sidebar, open "Connectors".
3. Click the "GitHub" card itself.
4. The project/repository linking action is inside that GitHub connector screen, not inside Project settings.

Mobile
1. Open Chat mode.
2. Tap the "..." menu.
3. Open "Connectors".
4. Tap "GitHub".
5. Use the project/repository linking option from there.

- "Connected to @Sunprem06" means your GitHub account is already connected.

- In this UI, Project settings does not contain the GitHub link action.
