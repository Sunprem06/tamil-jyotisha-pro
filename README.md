# Tamil Jothidam Pro

தமிழ் ஜோதிடம் மற்றும் கோவில் தகவல் தளம்

- 134+ temples across India and world
- 9 Navagraha Sthalams with planetary remedies
- 108 Divya Desams, 12 Jyotirlingas
- Interactive temple map with geo navigation
- Pilgrimage Passport tracker
- Daily spiritual updates in Tamil

## Tech Stack
- React 18 + Vite 5 + TypeScript
- Tailwind CSS + shadcn/ui
- Lovable Cloud (Supabase) backend
- Leaflet for maps

## Development
```bash
bun install
bun run dev
```

## Build
```bash
bun run build
```
